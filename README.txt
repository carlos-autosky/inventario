OPCIONES DE INSTALACIÓN:
━━━━━━━━━━━━━━━━━━━━━━━━

OPCIÓN 1 — INSTALADOR AUTOMÁTICO (recomendado)
  → Doble clic en: INSTALAR.bat
  • Verifica si Python está instalado
  • Instala todas las dependencias automáticamente
  • Crea un acceso directo en el Escritorio
  • Inicia el sistema

OPCIÓN 2 — USO DIRECTO
  → Doble clic en: run.bat
  • Instala dependencias si faltan
  • Inicia el sistema directamente

OPCIÓN 3 — CREAR EJECUTABLE .EXE
  → Doble clic en: CREAR_EJECUTABLE.bat
  • Compila un .exe autónomo (no requiere Python)
  • El .exe se guarda en la carpeta "dist\"
  • Funciona en cualquier PC con Windows 10/11
  • Tiempo de compilación: 3-5 minutos

REQUISITOS:
━━━━━━━━━━━
  • Windows 10 / 11 (64 bits)
  • Python 3.10 o superior (se descarga gratis en python.org)
  • Conexión a internet (solo para la primera instalación)

MÓDULOS INCLUIDOS:
━━━━━━━━━━━━━━━━━━
  • Inventario por Bodega
  • Detalle por SKU
  • SKU x Bodega
  • Muestras por Cliente (con reportes PDF/HTML)
  • Análisis de Período
  • Rotación y Compras (escenarios marítimo/aéreo)
  • Histórico de Compras
  • Kardex (con exportación PDF/Excel/HTML)
  • Toma Física (con plantilla Excel automática)
  • Log de actividad

SOPORTE DE FORMATOS:
━━━━━━━━━━━━━━━━━━━━
  • Entrada: Excel (.xlsx, .xls)
  • Exportación: Excel, PDF, HTML
